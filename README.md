# 依赖
## init Vuex
`npm i vuex@3 `
## init axios
`npm i vue-axios`
## init router
`npm i vue-router@3`
## init less
`npm i less less-loader@7.3.0`
## init mockjs mock数据
`npm i mockjs`
## install nprogress 加载进度条插件
`npm i nprogress`
## install swiper 轮播图
`npm i swiper@5`
## install iview 
`npm install iview`
